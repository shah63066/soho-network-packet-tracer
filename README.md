# SOHO Network Design (Cisco Packet Tracer)

## 📌 Project Overview
This project demonstrates the design and implementation of a Small Office Home Office (SOHO) network using Cisco Packet Tracer.

## 🛠️ Technologies Used
- Cisco Packet Tracer
- Networking Concepts (IP Addressing, VLAN, DHCP, Routing)

## ⚙️ Features
- Configured IPv4 addressing and subnetting
- Implemented VLANs for network segmentation
- Configured DHCP for automatic IP assignment
- Established routing between networks
- Basic network security configuration

## ▶️ How to Run
1. Download the `.pkt` file
2. Open in Cisco Packet Tracer
3. Run simulation mode to test connectivity
